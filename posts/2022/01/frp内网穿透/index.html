Frp内网穿透

内网传统配置简单记录

要求:拥有一台具有公网ip的服务器.

步骤：

1、下载frp安装包

进入github frp下载后解压。

2、配置服务器端

修改frps.ini文件，配置如下内容：

[common]
bind_port = 7000
token = 123 #可以自定义

3、配置客户端

在内网需要被远程的电脑修改frpc.ini文件：

[common]
server_addr = server_ip
server_port = 7000
token = 123 #与服务器端保持一致
[ssh]
type = tcp
local_ip = 127.0.0.1
local_port = 22
remote_port = 6000

4、开启防火墙

服务器和客户端均开启防火墙，开放上面配置的端口，如：

#CentOS
systemctl start firewalld
systemctl enbale firewalld firewall-cmd --add-port=6000/tcp --permanent
firewall-cmd --add-port=22/tcp --permanent
firewall-cmd --add-port=7000/tcp --permanent
firewall-cmd --reload
#Ubuntu
systemctl start ufw
systemctl enable ufw
ufw allow ssh
ufw status

5、设置frpc开机自启

vim /etc/systemd/system/frpc.service

[Unit]
Description=Frp Client
After=network.target
Wants=network.target

[Service]
Restart=on-failure
RestartSec=5
ExecStart=/opt/frp/frpc -c /opt/frp/frpc.ini#这里填写frp程序及配置文件对应的路径。

[Install]
WantedBy=multi-user.target

编辑完之后systemctl daemon-reload;

使用systemctl命令对frpc.service进行操作。

6、测试连接

使用ssh -p 6000 username@your_server_ip命令进行连接。

注意：

1、客户端与服务器端的frp版本、端口等需保持一致。

2、防火墙开启后端口是否开放。
